# 🚀 StudyBuddy

**An AI-powered collaborative learning platform built with the MERN Stack.**

StudyBuddy helps students stay consistent, organize study resources, track progress, collaborate in real time, and get instant AI-powered assistance during learning sessions.

---

## ✨ Key Features

- 🔐 JWT Authentication & Protected Routes
- 📊 Personal Learning Dashboard & Analytics
- 📅 Daily Study Tracker with Streak System
- 📝 Notes & Folder Management
- 📁 Study Material Storage
- 🤖 AI Learning Assistant
- 👤 User Profile Management
- 🏠 Collaborative Study Rooms
- 💬 Realtime Group Chat (Socket.io)
- 🎨 Shared Whiteboard for Discussions
- 🤖 AI Assistant Inside Study Rooms

---

## 🛠 Tech Stack

**Frontend:** React.js, Vite, Tailwind CSS, React Router

**Backend:** Node.js, Express.js

**Database:** MongoDB Atlas, Mongoose

**Authentication:** JWT, bcryptjs

**Storage:** Cloudinary

**Realtime Communication:** Socket.io

**AI Integration:** Google Gemini API

**Deployment:** Vercel, Render, MongoDB Atlas
